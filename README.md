This file is the mod for Sims 3. 
The file makes changes in DreamAndPromisesNodes.xml in GameplayData.package and will conflict with another files that changing this xml.
Install this file in Electronic Arts\The Sims 3\Mods\Packages
This mod fix the bug, when your sim try to find stars from telescope, and when sim wants to find more the 1 star, count is not progressing.

Этот файл - мод для Sims 3.
Файл вносит изменения в DreamAndPromisesNodes.xml из GameplayData.package и будет конфликтовать с другими файлами, вносящими те же изменения.
Установите мод как обычно в папку Electronic Arts\The Sims 3\Mods\Packages
Этот мод устраняет ошибку, когда сим пытается искать звёзды в телескоп, и когда сим хочет найти больше одной звезды - счётчик звёзд не продвигается.
